## 1. معرفی سرویس

**Shixon.com** یک پلتفرم فروشگاهی آنلاین مدرن و کامل است که با تمرکز بر تجربه کاربری فوق العاده، سرعت بالا و امنیت کامل طراحی و پیاده سازی شده است. این سیستم به عنوان یک راهکار تجارت الکترونیک کامل، تمامی نیازهای فروشندگان و خریداران را پوشش می دهد.

✅ این پروژه بر پایه فریم ورک ASP.NET MVC توسعه داده شده و دارای معماری پایدار، قابل توسعه و نگهداری است.

✅ سیستم در حال حاضر شامل ماژول های اصلی سبد خرید، پرداخت، مدیریت کاربران، مدیریت محصولات و سیستم گزارش گیری می باشد.

## 3. امکانات اصلی سرویس

### ✨ امکانات برای کاربران خریدار

- سیستم جستجوی پیشرفته و فیلتر محصولات
- سیستم دسته بندی چند سطحی محصولات
- صفحه جزئیات محصول با گالری تصویر و مشخصات کامل
- سیستم سبد خرید ذخیره شده
- سیستم نظرات و امتیازدهی کاربران
- لیست علاقه مندی ها
- پیگیری سفارشات زنده
- تاریخچه کامل خریدها

### ⚙️ امکانات برای مدیریت سیستم

- پنل مدیریت کامل و کاربرپسند
- مدیریت محصولات، دسته بندی ها و برندها
- مدیریت سفارشات و وضعیت سفارش
- سیستم تخفیف و کوپن ها

### 🔒 سیستم امنیتی و پرداخت

- رمزنگاری کامل اطلاعات حساس
- سیستم جلوگیری از تقلب و کلاهبرداری
- لاگ کامل تمام عملیات سیستم
- محافظت در برابر حملات رایج وب

## 4. مسیر کاربر (User Flow)

```
🔹 بازدید کننده وارد سایت می شود
    │
    ├─► مرور دسته بندی ها / جستجوی محصول
    │       │
    │       └─► مشاهده لیست محصولات
    │               │
    │               └─► ورود به صفحه جزئیات محصول
    │
    ├─► افزودن محصول به سبد خرید
    │       │
    │       └─► مشاهده و ویرایش سبد خرید
    │
    │
    ├─► تکمیل اطلاعات تحویل و آدرس
    │       │
    │       └─► انتخاب روش ارسال
    │
    ├─► انتخاب روش پرداخت
    │       │
    │       └─► پرداخت آنلاین ► انتقال به درگاه پرداخت
    │             │
    │             └─► تایید تراکنش و بازگشت به سایت
    │
    │
    │
    └─► ثبت نهایی سفارش
            │
            ├─► نمایش فاکتور سفارش
            ├─► ارسال پیامک تایید به کاربر
            └─► نمایش صفحه موفقیت آمیز بودن سفارش
```

## 5. معماری فنی و پلتفرم

این پروژه با استفاده از تکنولوژی های زیر توسعه داده شده است:

- **Backend**: C# / ASP.NET MVC 5
- **Database**: Microsoft SQL Server / Entity Framework
- **Frontend**: Bootstrap 4, jQuery, Vanilla JavaScript
- **Payment Gateway(s)**: Rubika
- **Hosting**: Windows Server / IIS
